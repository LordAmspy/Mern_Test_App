# MERN Backend (Day 1–4)

## Features
- Express server
- Routes & Controllers
- Async/Await (Day 3)
- MongoDB with Mongoose (Day 4)

## Setup

1. Install dependencies:
npm install

2. Add .env file:
MONGO_URI=your_connection_string

3. Run:
npm run dev

## APIs

GET /api/users
POST /api/users

Body:
{
  "name": "Test",
  "email": "test@gmail.com"
}
